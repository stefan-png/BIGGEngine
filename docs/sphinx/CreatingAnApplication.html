Creating An Application¶

To make an application, you need an instance of an implementation of ContextI. You can implement one yourself, or use the provided GLFWContext.

After creating a context, you should set up all your callbacks. This is done using ContextI::subscribe.

bool BIGGEngine::ContextI::subscribe(int8_t priority, EVENT_CALLBACK_FUNC func)

Pass a callback to listen to events.

Note

Negative priorities ( INT8_MIN to -1 ) are reserved for internal use!

Parameters

  • priority – Priority for recieving events. Lower priority is called first.

  • func – Callback function. It has type EVENT_CALLBACK_FUNC 

    // this is an example callback
bool myCallback(Events* event) {
 if(event->m_type == Events::EventType::MouseDown) {
     MouseDownEvent* mouseEvent = static_cast<MouseDownEvent*>(event);
     std::cout << mouseEvent->x << ", " << mouseEvent->y << '\n';
 }
}

Returns

true if event is handled and should not be propogated to other callbacks

Returns

false if event is not handled and should be propogated to other callbacks
